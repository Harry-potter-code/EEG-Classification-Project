# EEG-Classification-Project
# 基于泰勒多项式展开的神经网络脑电信号分类

本项目旨在利用改进的神经网络实现对脑电信号（EEG）的高效分类。

## 核心内容
- **数学模型**: 采用泰勒多项式特征展开（Taylor Polynomial Feature Expansion）增强特征提取能力。
- **优化算法**: 自定义梯度更新函数（Gradient_renewal），支持衰减学习率。
- **实验环境**: Python, PyTorch, NVIDIA RTX 4060 (8GB VRAM)。
